## 🚀 Overview
This repository showcases the **Front-End Design and Development** of the official landing page for the **C.O.R.E System** (a Python-based, context-aware AI desktop assistant for Linux). 

As a **Full-Stack Developer**, I built this promotional website to reflect the futuristic, cyberpunk nature of the underlying desktop application. The goal was to create a highly optimized, visually striking web experience without relying on heavy frameworks.

## ✨ Front-End Highlights
- **Framework-Free Performance:** Built entirely with **Vanilla HTML, CSS, and JavaScript**. Zero dependencies, no React, and no build steps required.
- **Glassmorphism & Neumorphism:** Custom-designed UI cards with transparent backdrops, animated glowing borders, and smooth hover states using Tailwind CSS.
- **Interactive 3D Elements:** A lightweight, customized `particles.js` instance engineered to run smoothly on mobile and desktop without draining CPU resources.
- **Responsive Architecture:** Fully optimized for all screen sizes with intelligent CSS Flexbox/Grid layouts.
- **Cyberpunk Typography & Glitch Effects:** Pure CSS keyframe animations to create terminal-like glitch effects on typography.

## 🛠️ Tech Stack
- **Structure:** HTML5
- **Styling:** Tailwind CSS (via CDN) + Custom CSS Keyframes.
- **Logic:** Vanilla JavaScript.
- **Icons:** Lucide Icons.
